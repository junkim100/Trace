# Trace

A second brain built from your digital activity.

Trace is a macOS app that continuously observes your digital activity, converts it into durable Markdown notes, stores typed relationships in a local graph, and enables time-aware chat and search over your past.

## Features

- **Automatic capture** - Screenshots, active apps, window titles, URLs, now playing, location
- **Hourly notes** - Markdown summaries generated every hour from your activity
- **Daily revision** - Notes refined with full-day context, entities normalized
- **Typed relationship graph** - Topics, apps, media, documents connected with weighted edges
- **Time-aware search** - Query by time range, topic, or activity type
- **Privacy-first** - All data stays local, raw artifacts deleted daily after processing

## Requirements

- macOS 14.0+
- Screen recording permission
- Accessibility permission (for window titles)
- Location services (optional)
